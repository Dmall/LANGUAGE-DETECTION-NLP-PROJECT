## Setup Instructions

1. Clone the Repository
```bash
git clone <repository-url>
cd LANGUAGE-DETECTION-NLP-PROJECT
```

2. Create Virtual Environment
```bash
conda create -p env python=3.10 -y
```

3. Install Dependencies
```bash
pip install -r requirements.txt
```

4. Run Streamlit
```bash
streamlit run ui.py
```
